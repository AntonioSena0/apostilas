1. Enunciado

Você foi contratado para gerenciar uma biblioteca. Sua missão é criar moldes (classes) para livros e facilitar o registro deles no sistema. Você deverá criar 2 classes, uma para rodar o sistema (Main) e outra para o molde do livro (Livro).

2. Preparação

  • Ferramenta: Intellij ou alguma IDE que suporte o Java.
  • Ambiente: Certifique-se de que o servidor Java está instalado no seu computador.

3. Desafios de Prática

Parte 1: Preparando o desenvolvimento

  1. Crie a classe Main para poder executar o código.
  2. Crie a classe Livro para criar o molde que será utilizado.

Parte 2: Preparando o molde e criando o objeto

  1. Crie os atributos String title; String description; String author; no seu molde.
  2. Instancie a classe Livro criando o objeto livro na sua classe Main utilizando Livro livro = new Livro(); e defina os atributos.

Parte 3: Criando o sistema

  1. Agora que tudo está funcionando, faça um debug de tudo isso ao final do código.
  2. Crie mais 2 objetos a partir do nosso molde e faça um debug deles no console também.

Quer mais dificuldade?

  1. Faça um sistema utilizando o Scanner, onde o usuário poderá cadastrar livros e armazena-los em um ArrayList onde ele poderá visualizar todos os livros cadastrados quando quiser.

5. Orientações Finais

  • Logs de Erro: Caso ocorra do código, não rodar, leia o que apareceu no console e tente decifrar onde foi o problema.
  • Anotações: Analise todo o código feito e comente acima de cada linha, a função que ela traz.
