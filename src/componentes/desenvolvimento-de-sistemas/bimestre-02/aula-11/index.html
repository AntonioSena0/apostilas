Teoria & Conceitos

1. História

Nascida na década de 1960 como um paradigma para modelar software de forma mais próxima ao mundo real, usando objetos que combinam dados e comportamentos. Os noruegueses Kristen Nygaard e Ole-Johan Dahl criaram a linguagem Simula em 1967, no Centro Norueguês de Computação, inicialmente para simulações de eventos discretos como explosões de navios. Nessa época eles já haviam introduzido conceitos fundamentais como:

  • Classe: Categorias de objetos com atributos e comportamentos únicos (Imagine um molde, onde podem ser feitos objetos de maneira mais fácil a partir dele, assim não precisando reconstruir o mesmo várias vezes)
  • Objetos: uma entidade que representa uma instância concreta de uma classe, ou seja, um “exemplar” específico com dados e comportamentos bem definidos.
  • Herança: Superclasses e subclasses, onde as subclasses herdam atributos e ações das superclasses (imagine como pai e filho, o filho herda algo do pai).

2. Analogia

Imagine que todas as linguagens antes da Simula, fizessem tudo na mão, como se toda a vez que você fosse navegar, tivesse que construir cada navio do zero - martelo, madeira, parafuso. Já com a Simula isso mudou, porque agora, toda a vez que precisar de um navio, você já tem um molde pronto (classe) e não precisa construir ele do zero.

Exemplo de Código

1. Montando classes

      
        class Carro {  // Classe (molde)
          String modelo;
          String cor;  // Atributo (estado)

          void acelerar() {  // Comportamento (método)
              System.out.println("Vrum!");
          }
        }

2. Criando objetos

      
        public class Main{

          public static void main(String[] args){

            //Nosso primeiro objeto (utilizamos new para instanciar uma classe)
            Carro carro = new Carro();

            /*
              Para acessarmos os atributos das nossas classes:
              Objeto.NomeDoAtributo
            */

            //Definindo o modelo do nosso carro (Objeto)
            carro.modelo = "Volkswagen Polo";
            //Definindo a cor do nosso carro (Objeto)
            carro.cor = "Preto";

            //Exemplo de debug
            System.out.println("Nosso objeto feito a partir do molde carro");
            System.out.println("Modelo: " + carro.modelo);
            System.out.println("Cor: " + carro.cor);

          }

        }

Recapitulando:

  • Classe: Classe é tudo aquilo que é criado para ser utilizado como molde. Ela será reutilizada várias vezes com o intuito de deixar o código mais simples e enxuto

  • Objeto: Objeto é tudo aquilo que é criado a partir de uma classe (toda instância de uma classe), com dados e ações próprias

  • Atributo: Uma variável associada a uma classe ou a seus objetos (instâncias), que armazena os dados ou características (estado) do objeto.
