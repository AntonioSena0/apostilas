1. Enunciado

Você foi contratado por uma instituição de ensino para automatizar a seleção de candidatos a uma bolsa de estudos integral. O sistema deve receber os dados acadêmicos e socioeconômicos de um aluno e informar, através de valores lógicos (true ou false), se ele atende aos critérios de diferentes modalidades de auxílio.

2. Preparação

  • Ambiente: Utilize uma IDE como VS Code (com extensão Java), IntelliJ ou um compilador online (ex: OnlineGDB).
  • Conhecimento Prévio: Requer conhecimento sobre declaração de variáveis (double, int, boolean) e comandos de entrada/saída (Scanner e System.out.println).
  • Documentação: Consulte a tabela de precedência de operadores (Parênteses > Relacionais > NOT > AND > OR) para estruturar as expressões.

3. Código Base (Exemplo de Demonstração)

Este código demonstra como capturar dados e realizar uma comparação simples utilizando operadores relacionais. 

import java.util.Scanner;

public class AnaliseBolsa {
    public static void main(String[] args) {
        Scanner leitor = new Scanner(System.in);

        // Entrada de dados
        System.out.print("Digite a média do aluno (0-10): ");
        double media = leitor.nextDouble();
        
        System.out.print("Digite a frequência do aluno (0.0-1.0): ");
        double frequencia = leitor.nextDouble();

        // Exemplo básico: Elegível para Bolsa Desempenho (Média >= 9.0)
        boolean bolsaDesempenho = media >= 9.0;

        System.out.println("Elegível para Bolsa Desempenho? " + bolsaDesempenho);
        
        leitor.close();
    }
}

4. O Desafio

Expanda o programa acima para avaliar duas novas modalidades de auxílio, utilizando operadores lógicos (&&, ||, !) para combinar as condições:

  • Bolsa Científica: O aluno deve ter uma média igual ou superior a 8.0 E uma frequência de no mínimo 80% (0.80).
  • Auxílio Social: O aluno deve ter uma rendaFamiliar menor que R$ 2.000,00 OU ser integrante de um programa social (boolean pertenceProgramaSocial).
  • Bolsa Integral (Ouro): Para ganhar esta bolsa, o aluno deve atender ao critério da Bolsa Científica (item 1) E ao critério do Auxílio Social (item 2).

Requisitos técnicos:

  • Crie as variáveis necessárias para capturar a rendaFamiliar e se pertenceProgramaSocial.
  • Utilize parênteses para garantir que a lógica da "Bolsa Integral" seja avaliada corretamente, respeitando a hierarquia dos conectivos lógicos.
  • O programa deve exibir o resultado true ou false para cada uma das três novas modalidades.

5. Orientações Finais

  • Indentação: Mantenha o código organizado com recuos para facilitar a leitura.
  • Tipagem: Lembre-se que em Java o operador de igualdade é == e o de atribuição é =.
  • Teste de Mesa: Antes de rodar, tente prever o resultado para um aluno com Média 8.5, Frequência 0.85 e Renda de R$ 5.000,00.
