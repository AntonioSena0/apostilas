Enunciado

Neste desafio, você recebeu a interface de um Sistema de Biblioteca Técnica desenvolvida em PHP e Bootstrap. O arquivo principal, index.php, contém toda a lógica de exibição e o formulário de cadastro, mas o sistema não funcionará enquanto a classe que representa o objeto "Livro" não for criada.

Sua tarefa é estudar o código do index.php para deduzir a estrutura necessária da classe. Você deve descobrir, apenas lendo o código fornecido:

  • Quais dados o construtor da classe deve estar preparado para receber e em qual ordem?
  • Quais métodos a tabela HTML tenta chamar dentro do loop foreach para exibir as informações na estante?

Preparação

  1. Certifique-se de estar utilizando o PHP 8.1 ou superior para aproveitar as funcionalidades modernas da linguagem.
  2. Crie uma pasta para o exercício no seu ambiente local (XAMPP ou diretório de sua preferência).
  3. Salve o código abaixo como index.php.
  4. Crie um arquivo novo e vazio chamado Livro.php na mesma pasta.

O Desafio: Criar Livro.php

Você deve escrever o código do arquivo Livro.php começando do zero. Regras de implementação:

  • A classe deve utilizar o Paradigma Orientado a Objetos.
  • Encapsulamento: Mantenha os dados internos protegidos para que o mundo externo só consiga acessá-los através dos métodos públicos necessários.
  • Identidade Própria: Utilize a palavra-chave reservada para se referir aos dados da instância atual que está sendo processada no momento.
  • Tudo está no index.php: Não invente nomes de variáveis ou funções. Se o index.php tenta chamar uma função ou enviar um dado, sua classe deve ter exatamente esse nome e estrutura para que a conexão entre os arquivos funcione perfeitamente.

Orientações Finais

  • Estudo Atento: Todos os nomes de propriedades e métodos que sua classe precisa ter estão descritos e utilizados diretamente no arquivo index.php. Sua missão é garantir que a classe seja o "encaixe" perfeito para o que a interface já espera consumir.
  • Rodando o Projeto: Para testar sua aplicação, utilize o servidor embutido do PHP. Abra o terminal na pasta do projeto e execute:
    php -S localhost:8000
  • Validação: Abra http://localhost:8000 no navegador. Se o formulário cadastrar os dados e a tabela exibir as informações corretamente sem erros de sistema, sua engenharia reversa foi concluída com sucesso!
  • Dica Profissional: Ao criar o construtor, considere utilizar a técnica de Promoção de Propriedades do PHP 8 para tornar seu código mais limpo e eficiente.
